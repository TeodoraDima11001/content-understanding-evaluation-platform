# Content Understanding Evaluation Platform

## Overview
Production-style Data Science system inspired by Spotify Content Platform.

This project focuses on:
- Content taxonomy systems
- Annotation quality evaluation (human + LLM)
- Content classification systems
- SQL + dbt analytics pipelines
- Experimentation frameworks
- Metrics and decision systems

## Problem
Large-scale content platforms must ensure:
- Correct classification of content
- High-quality metadata
- Reliable LLM annotations
- Safe and effective recommendations

## Goal
Build an end-to-end evaluation system that:
- Measures content understanding quality
- Evaluates annotation systems
- Supports data-driven decisions
- Mimics production DS systems at Spotify scale
